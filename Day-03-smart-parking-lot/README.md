# Day 3 - Smart Parking System

## Project Overview

This project is a Smart Parking System built using Arduino Uno, IR Sensors, Servo Motor, and Buzzer.

The system automatically detects vehicle entry and exit, controls the parking gate using a servo motor, and maintains parking count logic.

---

## Components Used

* Arduino Uno
* 2 IR Sensors
* Servo Motor
* Active Buzzer
* Breadboard
* Jumper Wires

---

## Features

* Automatic entry and exit detection
* Servo-controlled parking gate
* Real-time parking count
* Buzzer alerts
* Embedded automation logic

---

## Learning Outcome

* Multi-sensor interfacing
* Servo motor control
* Embedded automation systems
* Event-driven logic implementation
* Real-world parking management concepts
