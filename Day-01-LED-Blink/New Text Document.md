# Day 1 - Arduino LED Blink Project

## Project Overview
This was my first embedded systems beginner project using Arduino Uno.

The project focuses on controlling LEDs using Arduino programming and understanding basic embedded system concepts.

---

## Components Used
- Arduino Uno
- LEDs
- Resistors
- Breadboard
- Jumper Wires

---

## Features
- LED blinking sequence
- Basic output control
- Timing using delay()

---

## Learning Outcome
- Arduino IDE basics
- Digital output control
- Circuit connections
- Embedded systems fundamentals
