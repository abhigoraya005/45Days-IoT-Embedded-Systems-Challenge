# Day 2 - Bidirectional Visitor Counter

## Project Overview
This project is a bidirectional visitor counter using Arduino Uno, 2 IR sensors, and a buzzer.

It detects people entering and exiting and updates the inside count accordingly.

---

## Components Used
- Arduino Uno
- 2 IR Sensors
- Active Buzzer
- Breadboard
- Jumper Wires

---

## Features
- Entry and exit detection
- Live people counting
- Buzzer indication
- Serial Monitor output

---

## Learning Outcome
- Sensor interfacing
- Embedded systems logic
- Digital input handling
- Counter implementation
